# ALLscan
一款基于GO语言开发的图形化简单EXP管理及测试工具。

ALLscanV1.0版本上线预计在10月份之前

为了方便POC记录与漏洞管理，我尝试开发这款简单的工具，功能暂时不完善。等待2.0版本进行补充完善
